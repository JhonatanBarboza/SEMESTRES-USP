# Redes de Computadores

Este repositório reúne os projetos e atividades desenvolvidos na disciplina de **Redes de Computadores** da Universidade de São Paulo (USP/ICMC). O conteúdo contempla os principais conceitos de redes de comunicação, incluindo os modelos OSI e TCP/IP, protocolos de comunicação, roteamento, endereçamento IP, transporte de dados, redes locais e simulações de infraestrutura, consolidando conhecimentos teóricos e práticos na área.
