# Bases de Dados

Este repositório reúne os projetos e atividades desenvolvidos na disciplina de **Bases de Dados** da Universidade de São Paulo (USP/ICMC). O conteúdo contempla os principais conceitos de bancos de dados, incluindo modelagem conceitual com o Modelo Entidade-Relacionamento (MER), modelo relacional, mapeamento MER-relacional, álgebra relacional, normalização, linguagem SQL, transações e desenvolvimento de sistemas utilizando Sistemas Gerenciadores de Banco de Dados (SGBDs), consolidando conhecimentos teóricos e práticos na área.
