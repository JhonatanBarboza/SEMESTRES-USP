# SEMCOMP

Plataforma web desenvolvida para a **Semana da Computação (SEMCOMP)**, permitindo o gerenciamento de eventos, participantes, atividades e conteúdos da organização. O sistema utiliza uma arquitetura Full Stack moderna, integrando front-end, back-end, banco de dados e APIs para oferecer uma experiência eficiente tanto para os organizadores quanto para os participantes.
