# Engenharia de Software

Este repositório reúne os projetos, exercícios e estudos desenvolvidos na disciplina **SSC0130 – Engenharia de Software** da Universidade de São Paulo (USP/ICMC). O conteúdo aborda as principais etapas do desenvolvimento de software, incluindo levantamento e especificação de requisitos, modelagem, arquitetura, gerenciamento de projetos, controle de versão, garantia da qualidade, testes, manutenção e metodologias ágeis, consolidando conhecimentos teóricos e práticos para o desenvolvimento de sistemas de software.
