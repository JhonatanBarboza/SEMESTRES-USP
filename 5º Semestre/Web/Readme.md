## Introdução ao Desenvolvimento Web

Este repositório reúne os projetos desenvolvidos na disciplina **SCC0219 – Introdução ao Desenvolvimento Web** da Universidade de São Paulo (USP/ICMC). Os projetos exploram o desenvolvimento de aplicações web modernas, aplicando conceitos de front-end, back-end, bancos de dados, APIs REST, autenticação e interfaces responsivas, consolidando os conhecimentos adquiridos ao longo da disciplina.
